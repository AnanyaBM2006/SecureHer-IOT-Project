Title: 

IoT-Based Women’s Safety Emergency Alert System using ESP32

2. Abstract

Women’s safety has become a critical concern in many urban and rural areas. In emergency situations, 
victims often require a fast and simple method to send alerts and share their location with authorities 
or trusted contacts. This project presents an IoT-based Women’s Safety Emergency Alert System built using
 the ESP32 microcontroller. The system uses a capacitive touch sensor as a trigger mechanism embedded in a 
wearable device such as a necklace or bracelet. When the sensor is pressed, the ESP32 activates an emergency 
alert, displays the status on a 16×2 LCD screen, and generates simulated GPS coordinates (latitude and longitude). 
These coordinates represent the victim’s location and can be logged into a cloud platform such as Google Sheets for
monitoring and tracking purposes. The prototype demonstrates how low-cost IoT hardware can be used to build a quick-response 
safety system that can be expanded in the future to include real GPS modules, SMS alerts, and mobile notifications.


3. Problem Statement

In many emergency situations, women may not have sufficient time or ability to call for help using a mobile phone. 
Traditional safety methods may require multiple steps, which can delay alert generation during critical moments. 
There is a need for a simple, fast, and wearable safety device that allows a user to trigger an emergency alert instantly. 
The system should be capable of generating location information and logging the event for further action. The challenge is to 
design a low-cost IoT prototype that can quickly activate an alert and share location data when a trigger is pressed.


4. Objectives


• To design a wearable emergency alert system for women’s safety
• To use a touch sensor as a quick and easy emergency trigger mechanism
• To implement the system using an ESP32 microcontroller
• To display alert messages on a 16×2 LCD display
• To generate simulated GPS latitude and longitude values for location tracking
• To log emergency alerts and location data into a cloud platform (Google Sheets)
• To demonstrate a scalable IoT architecture for real-time safety monitoring

8. Implementation (Git)

The system was implemented using Arduino-based embedded programming. The ESP32 microcontroller continuously 
monitors the touch sensor input. When the sensor detects a touch event, the microcontroller triggers the emergency
 alert logic. The LCD screen immediately displays an SOS alert message indicating that the emergency system has been 
activated. Simultaneously, the system generates simulated GPS coordinates using a random number generation method to 
represent a location. These coordinates are printed in the serial monitor and prepared for logging into a cloud database
 such as Google Sheets. The implementation can be extended by integrating a real GPS module and automated cloud logging 
through HTTP requests.
